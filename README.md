1 - open the new command prompt

#     npm install

---------------------------------------

2 - After installation completes, use the following commnad to start the application

#      npm start

  Local:  http://localhost:3000/

---------------------------------------

3 - open the new command prompt

#      npm run build

#      npm install -g serve
